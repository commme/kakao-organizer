# 카카오톡 채팅방 정리기 📒

**카톡에 쌓인 수백 개 메모·링크를 5분 만에 비워주는 웹 도구.**

🔗 바로 쓰기: **https://commme.github.io/kakao-organizer/**

---

## 이런 분께

- 카톡 "나에게 보내기"에 링크·메모가 수백 개 쌓여 있다
- 다시 찾기 귀찮아서 결국 안 본다
- 노션·엑셀로 옮기고 싶은데 하나씩 복사하긴 싫다
- AI 프롬프트·유튜브 링크·레시피를 카톡에 던져놓기만 한다

---

## 30초 요약

1. 카톡에서 대화 `.txt` 내보내기
2. 사이트에 파일 드래그
3. 자동 분류 결과 확인 → **⚡ AI 요약** 클릭 → 엑셀 다운로드 → 끝

데이터는 **전부 본인 브라우저에만 저장**. 서버로 안 보냅니다.

---

## 시작하기 (3단계)

### 1. 카톡에서 대화 내보내기

```
카카오톡 앱 → 채팅방 → 메뉴(☰) → 대화 내보내기 → 텍스트(.txt)
```

"나와의 채팅"이든 일반 채팅방이든 모두 됩니다.

### 2. 사이트 접속 → 파일 드래그

https://commme.github.io/kakao-organizer/

받은 `.txt` 파일을 화면에 끌어다 놓습니다.

### 3. 분류 → AI 요약 → 다운로드

| 버튼 | 동작 |
|------|------|
| ⚡ **AI 요약** | 현재 보이는 항목을 한 번에 요약 + 엑셀 자동 다운로드 |
| 📤 **내보내기** | 보관함 전체 / AI 요약만 / 프롬프트 모음 4가지 형식 |
| 📂 (카드별) | 카테고리 잘못 분류된 거 수정 → 도메인 자동 학습 |
| 🗑 (카드별) | 필요 없는 항목 휴지통으로 |

## 주요 기능

### 자동 분류 (16 카테고리)

| 영역 | 카테고리 |
|------|----------|
| 일·생산성 | AI/툴 · 개발 · 디자인 · 업무 · 커리어 |
| 정보·학습 | 학습 · 뉴스 · 블로그 |
| 관계·라이프 | SNS · 음식 · 여행 · 건강 · 취미 |
| 자산·소비 | 쇼핑 · 금융 |

도메인 매핑(예: `youtube.com` → 취미) + 키워드 매칭 + (선택) Gemini AI 분류 3단계.

### AI 요약 (선택)

Gemini API 키를 입력하면:
- **URL**: 페이지 내용을 읽고 **30자 이내**로 요약
- **텍스트 메모**: 핵심을 **50자 이내**로 요약

키 없어도 도메인 매핑만으로 80%는 알아서 분류됩니다.

### 프롬프트 자동 추출

긴 AI 프롬프트·가이드를 4종으로 자동 분류:
- 🎨 **이미지** (Midjourney, SDXL, Flux...)
- 🎬 **영상** (Sora, Veo, Runway, Kling...)
- 💻 **개발소스** (코드 블록, 함수, API)
- 📋 **가이드** (단계별 설명, 튜토리얼)

설정에서 분류 키워드를 직접 편집할 수 있습니다.

### 자동 학습

📂 버튼으로 카테고리 바꾸면 그 도메인은 다음부터 자동으로 해당 카테고리로 분류됩니다.

### 누적 업데이트

어제까지 정리한 항목은 그대로 두고, 오늘 내보낸 카톡만 추가됩니다.

## 내보내기 형식 4가지

1. **보관함 전체 (.xlsx)** — 카테고리별 시트로 분리된 완전한 엑셀
2. **AI 요약만 (.xlsx)** — 요약된 것만 깔끔하게
3. **프롬프트 (.md)** — 이미지/영상/개발/가이드 프롬프트만 마크다운으로
4. **프롬프트 (.xlsx)** — 같은 내용을 엑셀로

---

## Gemini API 키 받기 (무료)

1. https://aistudio.google.com/apikey 접속
2. "Create API key" 클릭
3. 받은 키(`AIza...`로 시작)를 **⚙️ 설정**에 붙여넣기
4. 저장

무료 티어로 하루 수백 개 요약 가능. 키는 본인 브라우저에만 저장되고 외부로 안 나갑니다.

## 자주 묻는 질문

**Q. 설치해야 하나요?** 아뇨. 링크 접속만 하면 됩니다.

**Q. 로그인 필요해요?** 아뇨. 계정 전혀 없습니다.

**Q. 데이터는 어디 저장되나요?** 본인 브라우저의 localStorage. 사이트를 닫아도 유지되고, 다른 사람은 못 봅니다.

**Q. 카톡 대화 내용이 외부로 나가나요?** **아니요.** AI 요약을 켤 때만 해당 URL/텍스트가 Gemini로 전송됩니다.

**Q. 몇 개까지 처리 가능?** 1,000개 정도까지 무리 없이 동작.

**Q. 모바일에서도 쓸 수 있나요?** 네, 반응형입니다. 단, 카톡 대화 내보내기는 PC가 편합니다.

**Q. 잘못 분류된 게 많아요.** 📂 버튼으로 바꾸면 자동 학습됩니다. 설정 → "학습 매핑 보기"에서 확인/삭제 가능.

**Q. 전부 초기화하려면?** 설정 → "전체 데이터 초기화" 버튼.

## 보안·개인정보

| 항목 | 상태 |
|------|------|
| 로그인/계정 | ❌ 없음 |
| 서버 저장 | ❌ 없음 (모두 브라우저 localStorage) |
| 외부 전송 | AI 요약 시에만 Gemini로 해당 항목만 전송 |
| API 키 보관 | 본인 브라우저에만 (브라우저 격리) |
| 외부 스크립트 | SheetJS CDN 1개만 허용 (CSP) |
| XSS 방지 | 모든 사용자 입력 `textContent` 처리 |
| 악성 URL | `http/https`만 허용 (`javascript:` 차단) |

## 단축 기능

- 카드 클릭 → URL 새 탭 열기
- ☑ 선택 모드 → 여러 항목 한꺼번에 카테고리 변경/삭제/AI 요약
- 카테고리 chip 클릭 → 해당 카테고리만 필터링
- 프롬프트 sub-chip → 현재 카테고리 안에서 이미지/영상/개발/가이드만
